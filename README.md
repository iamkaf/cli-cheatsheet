# cli-cheatsheet

A minimal Astro site for `cli.kaf.sh` with a raw HTML cheatsheet for:

- zoxide
- fd
- fzf
- rg
- jq

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run check`
- `npm run deploy`

## Deploy

This project deploys to Cloudflare Workers static assets via `wrangler deploy` using `wrangler.jsonc`.
